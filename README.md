# IronWill

Интерактивный веб-проект фитнес-клуба с адаптивным интерфейсом и пользовательскими виджетами.

## О проекте

Проект разработан как учебное приложение с акцентом на практику JavaScript и работу с интерактивными UI-компонентами без использования фреймворков.

В приложении реализованы пользовательские калькуляторы, интерактивные элементы интерфейса, работа с JSON-данными, localStorage и SVG.

## Функциональность

* адаптивная верстка;
* интерактивная SVG-карта мышц;
* генератор тренировок на основе JSON-данных;
* калькуляторы:

  * BMR;
  * нормы воды;
  * нормы белка;
  * сожжённых калорий;
  * стоимости абонемента;
* квиз для подбора тарифа;
* пользовательские чек-листы с сохранением состояния через localStorage;
* галерея;
* кастомные select-компоненты;
* анимация счетчиков при скролле;
* виджет случайных советов;
* кнопка scroll-to-top;
* интеграция карты через MapLibre;
* SVG sprite для иконок.

## Технологии

* HTML5
* CSS3
* JavaScript (ES6)
* BEM
* Fetch API
* LocalStorage
* IntersectionObserver

## Что было реализовано в процессе

* работа с асинхронными запросами (`fetch`, `async/await`);
* динамическое обновление DOM;
* работа с JSON-структурами;
* обработка пользовательских событий;
* создание переиспользуемых UI-элементов;
